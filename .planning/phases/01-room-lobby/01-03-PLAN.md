Create src/components/PlayerCard.tsx (client component):
- Props: player (Player | null), isCurrentUser (boolean), onToggleReady (() => void)
- If player is null: render empty slot — dashed border, "Waiting for player..." text, muted colors
- If player exists:
  - Show nickname prominently
  - Host indicator: crown or star icon (use Unicode or SVG) next to name if player.isHost
  - Ready status: card background changes color when ready (e.g., green tint when ready, neutral when not)
  - "READY" / "NOT READY" label visible on card
  - If isCurrentUser: show a "Ready" toggle button on the card. Button text: "Ready!" when not ready, "Cancel" when ready
- Animation: fade-in when player joins (use Tailwind transition classes)

Create src/components/Countdown.tsx (client component):
- Props: count (number), onComplete (() => void)
- Full-screen overlay (fixed, z-50, semi-transparent dark background)
- Large centered number (3, 2, 1) with scale-in animation
- When count reaches 0 or below, call onComplete
- Style: retro arcade feel — large bold number, maybe pulsing effect

Create src/components/Lobby.tsx (client component):
- Props: room (Room), currentPlayerId (string)
- Header section:
  - Room code displayed large (e.g., 3xl font, monospace): "ROOM: ABCD"
  - "Copy Code" button: copies code to clipboard, brief "Copied!" feedback
  - Shareable link displayed below: full URL like `{origin}/room/ABCD` with copy button
- Player grid: 2x2 CSS grid layout
  - Always render 4 PlayerCard slots
  - Fill with room.players data, rest are null (empty slots)
  - Pass isCurrentUser and onToggleReady to the current player's card
- Ready toggle: emit 'room:ready' with { ready: !currentReady } via socket
- Host Start button (below grid):
  - Only visible if current user is host
  - Enabled only when 2+ players in room have isReady === true
  - Disabled state: grayed out with tooltip "Need 2+ ready players"
  - On click: emit 'room:start' via socket
- Toast notifications: when a player joins/leaves, show a brief notification (use a simple toast component or just a timed div). "Player X joined!" / "Player X left"
- Listen for socket events:
  - 'room:updated': update room state
  - 'room:player-joined': show join toast
  - 'room:player-left': show leave toast
  - 'room:countdown': show Countdown overlay with the count value
  - 'room:game-start': for now, show "Game starting..." message (actual game comes in Phase 2)

Update src/app/room/[code]/page.tsx:
- Replace placeholder lobby with the full Lobby component
- Pass room state and currentPlayerId
- Manage room state from socket events
- Handle disconnect/reconnect gracefully
