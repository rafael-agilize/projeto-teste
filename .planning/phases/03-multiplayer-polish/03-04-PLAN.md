  - Use CSS @keyframes, triggered by adding a class
- Piece lock animation: brief flash/pulse on the piece cells when piece locks in place (~100ms)
- Use CSS animations and transitions (not JS-driven frame-by-frame) for performance
- All animation durations should feel snappy — this is arcade, not cinematic
