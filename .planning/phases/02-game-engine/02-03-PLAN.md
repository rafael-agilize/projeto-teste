Implement T-spin detection using the 3-corner rule:

**src/engine/tspin.ts:**
```typescript
export type TSpinResult = 'none' | 'tspin' | 'tspin-mini';

export function detectTSpin(
  board: BoardState,
  piece: Piece,
  lastActionWasRotation: boolean
): TSpinResult
```

**Detection algorithm:**
1. If piece type is not 'T', return 'none'
2. If lastActionWasRotation is false, return 'none'
3. Check 4 corner cells of the T piece's 3x3 bounding box
4. Count how many corners are occupied (out of bounds counts as occupied)
5. If 3+ corners occupied: 'tspin'
6. If 2 corners occupied AND last rotation used a wall kick: 'tspin-mini'
7. Otherwise: 'none'

Note: The function needs to know if a wall kick was used. Add `lastRotationUsedKick: boolean` parameter.

**Scoring table (per user decision -- Modern Tetris Guideline):**
- Base points by clear type and T-spin status (see behavior section)
- Combo bonus: added on top, 50 * combo * level
- Back-to-back: if current clear is tetris or any T-spin clear, AND backToBack flag is true, multiply base by 1.5
- Update backToBack: set to true if current clear is tetris or T-spin, false otherwise
- Update combo: increment if lines cleared > 0, reset to -1 if no lines cleared
